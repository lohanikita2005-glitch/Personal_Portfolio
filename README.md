#Personal_Portfolio
## Personal Portfolio Website

This project is developed as part of the **Future Interns – Full Stack Web Development Internship**  
(Task 1: Personal Professional Portfolio Website).

The purpose of this project is to create a clean, responsive, and professional
portfolio website to showcase personal skills, projects, and contact information.

---

## 📌 Project Objective

- Build a personal portfolio website  
- Showcase skills and projects  
- Provide a simple and user-friendly UI  
- Follow basic web development best practices  

---

## ✨ Features

- About Me section  
- Skills section  
- Projects showcase  
- Contact information  
- Responsive design for different screen sizes  

---

## 🛠 Technologies Used

- HTML5  
- CSS3  
- JavaScript
